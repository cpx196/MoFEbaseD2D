# Archived D2DMoE Framework

This directory retains the upstream D2DMoE framework. Its root modules are
`common.py`, `train.py`, `eval.py`, and `utils.py`; the corresponding
architectures, methods, scripts, visualization utilities, and data helpers are
kept alongside them with their original internal layout.

The current GPT-2 Dense, MoFE group-LR, and Upcycling workflow is
self-contained in `../../MoFE/` and does not import this archive.
