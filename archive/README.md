# Archive

This directory preserves repository content that is not required by the active
GPT-2 Dense, MoFE group-LR, and Upcycling workflow.

- `legacy_d2dmoe/`: the original D2DMoE framework, including its architectures,
  methods, visualization utilities, data helpers, scripts, and four framework
  modules (`common.py`, `train.py`, `eval.py`, and `utils.py`).
- `project_history/`: retired environment and launch files, upstream notes,
  method notes, the handoff document, the unused teaser asset, and historical
  early experiment results.

The active project implementation is in `../MoFE/`. The final 50k experiment
archive is in `../results/final_50k/`.
